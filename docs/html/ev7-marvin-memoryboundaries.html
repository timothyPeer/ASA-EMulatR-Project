EV7-Marvin

This is documented, but is not implemented.

 

EV7 (21364 + “Marvel” on-chip cross-bar & IO7 ASIC)

EV7 pulls the memory controller onto the processor die and extends the PA to 44 bits as well, but Marvel reserves the upper 1 TB of the memory half for inter-node routing and leaves PIO in the high 8 TB exactly as EV6—this is why Linux/Tru64 use the same constants.

reserved

CPU-visible semantics (PIO vs memory, ordering rules, PAL-TBIS behaviour) are unchanged from EV6—the only differences are the node-ID bits and which ASIC answers a given hose number.
