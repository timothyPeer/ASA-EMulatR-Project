128-bit Integer: dt_int128

A. 128-bit Integer: dt_int128

-----------------------------

 

The `dt_int128` class provides an explicit, architecture-neutral signed 128-bit integer as two 64-bit words (`high`, `low`). It supports construction from 64-bit values, two's complement arithmetic, bitwise logic, shift operations, signed multiplication, and overflow detection. Usage conforms to the Alpha MULQ, UMULH, and related quadword instructions.

 

Example usage:

```cpp

#include "dt_int128.h"

dt_int128 prod = dt_int128::multiply(a, b); // Full 128-bit multiply (signed)

quint64 resultLow = prod.low;

quint64 resultHigh = prod.high;

 

Downcasting:

To extract a 64-bit value, use low64() for the least significant bits or high64() for the upper 64 bits. Truncation from 128 to 64 bits does not perform saturation or sign-extension:

 

qint64 result64 = static_cast<qint64>(prod.low);

 

Overflow should be checked using dt_int128::hasSignedOverflow().
