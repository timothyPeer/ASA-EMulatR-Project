Prefetch Depth Configuration:

1. Default Values:

 

Prefetch Depth: 2 cache lines ahead (default)

Prefetch Distance: 128 bytes ahead (default)

 

2. Configuration Methods:

cpp// Set how many cache lines to prefetch ahead

integrator->setPrefetchDepth(4);        // Prefetch 4 cache lines ahead

 

// Set prefetch distance in bytes  

integrator->setPrefetchDistance(256);   // Look 256 bytes ahead

 

3. QSettings Integration:

Add to your config.ini:

ini[TlbCacheIntegration]

; Number of cache lines to prefetch ahead (default=2)

prefetchDepth=2

; Prefetch distance in bytes (default=128)  

prefetchDistance=128

 

4. Auto-Detection from Cache Controllers:

If your UnifiedDataCache has these properties, they'll be auto-detected:

cppclass UnifiedDataCache : public QObject {

   Q_OBJECT

   Q_PROPERTY(quint32 prefetchDepth READ getPrefetchDepth WRITE setPrefetchDepth)

   Q_PROPERTY(quint32 prefetchDistance READ getPrefetchDistance WRITE setPrefetchDistance)

   // ...

};

 

5. Prefetch Operation with Depth:

When a prefetch is triggered, it now prefetches multiple cache lines:

cpp// Prefetches cache lines at: address, address+64, address+128, address+192

// (for prefetchDepth=4 and 64-byte cache lines)

Typical Usage:

 

L1 Cache: Shallow prefetch (depth=1-2) for low latency

L2 Cache: Medium prefetch (depth=2-4) for balance

L3 Cache: Deep prefetch (depth=4-8) for high bandwidth
