Constant-Expressions

 

Considerations.

•Constants should contain a prefix denoting the region, register, type associated with the constant.

 

Constant

Description

SUPERPAGE_BASE

Base address: 0xFFFF_FFFF_8000_0000 (hex, 64-bit)

Is the canonical base for the Alpha AXP kernel's "superpage" region, where PALcode and OS map internal structures (PCR, TEB, panic handler, etc.) for each CPU.

RESTART_VECTOR

Refer to: RESTART_VECTOR Register

RESTART_VECTOR_UNRECOVERABLE_FAULT

Deposited to the RESTART_VECTOR register to indicate no restart is possible.

ASTEN_KERNEL_SHIFT

Kernel mode (0)

ASTEN_EXECUTIVE_SHIFT

Executive mode (1)

ASTEN_SUPERVISOR_SHIFT

Supervisor mode (2)

ASTEN_USER_SHIFT

User mode (3)

ASTEN_MASK

Bits 0-3 (0x0F)
