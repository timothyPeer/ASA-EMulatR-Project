ASA-EMulatR Initialization

 

 

Step

Action

1

SystemLoader loads system parameters from JSON config

2

Calls AlphaSMPManager::configureSystem(cpuCount, memoryMB, startPC)

3

AlphaSMPManager dynamically creates AlphaCPUs and shared memory system

4

CPU program counters (PC) initialized to 0x21000000 (or from config)

5

Shared AlphaMemorySystem used for RAM and ROM

6

Configures Serial ports, Network cards, Storage controllers from JSON

 

 

CPU ID

Starting PC

How to Handle

CPU 0

Always 0x21000000

Primary boot via SRM firmware

CPU 1..n

Start at 0x00000000 or idle loop

Until OS reassigns and sends a resume/start signal

Later (via OS)

OS will issue "start this CPU at PC=X"

Implement AlphaCPU::setPC() + resumeExecution()

 

initialization example

/*

for (int i = 0; i < cpuCount; ++i) {

    AlphaCPU* cpu = new AlphaCPU(i, memorySystem, this);

    

    if (i == 0) {

        cpu->setPC(0x21000000); // PALcode reset vector

    } else {

        cpu->setPC(0x00000000); // Idle or wait-for-interrupt loop

        cpu->haltExecution();   // Optional: Start halted

    }

 

    m_cpus.append(cpu);

}

 

// Simulate OS call to start CPU 2 at 0x100000

m_cpus[2]->setPC(0x00100000);

m_cpus[2]->resumeExecution();
