Show Device

Displays information for devices on the system.

 

Syntax

sh[ow] dev[ice] [dev_name]

 

Options

None

 

Argument

dev_name

Any adapter name (wildcarding is allowed). For example, show device dk* will display information on all SCSI devices on the system. If dev_name is omitted, the display shows all devices in the system.

 

Example

P00>>>show dev
