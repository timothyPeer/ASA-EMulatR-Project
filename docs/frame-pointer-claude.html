Frame Pointer

 

Project References:

AlphaCPU::stackFrame::framePointer

 

Usage:  

•Holds the base address of the current stack frame (in function/procedure calls)

•Used by PAL and OS to manage trap frames and stack unwinding

•Saved during traps to allow returning from interrupt context
