Property and Setting Reference

ASA EmulatR properties are configured using the following settings as a guide.

 

[System]

 

Key Name

Type

Description/Example

MemorySizeGB

(int)

Total system RAM, in GiB (e.g., 32)

hw-Model

(string)

Hardware model (e.g., ES40, AlphaServer 1000A)

hw-Serial-Number

(string)

Board or chassis serial

Coherency-Cache

(int)

L1 I-cache coherency size in KiB (typical: 2048)

Platform-Ev

(int)

Alpha EV family (4=EV4, 5=EV5, 6=EV6)

PTE-PageSize

(int)

MMU page size, bytes (power of two, e.g., 8192)

CPU_FREQUENCY_HZ

(int)

CPU frequency in Hz (e.g., 500000000)

 

[Rom]

Key Name

Type

Description/Example

clipperFileName

(string)

Clipper ROM binary

HostProcessorModuleFirmwareFile

(string)

Host processor firmware (HEX)

PCIBusModuleFirmWare

(string)

PCI bus firmware (HEX)

SystemModuleFirmwareFile

(string)

System module firmware (HEX)

IntelHexLoaderFile

(string)

Generic HEX loader

 

[CPU]

Key Name

Type

Description/Example

Processor-Count

(int)

Number of CPUs in SMP (e.g., 1, 2, 4, 8)

 

[Threads]

Key Name

Type

Description/Example

ThreadCount

(int)

Thread pool count for SMP/IO

 

[Session]

Key Name

Type

Description/Example

fName / logFileName

(string)

Log file path

logLevel

(int)

Log verbosity (0=info)

Method / updateMethod

(string)

File handling ("append", "truncate")

 

[CACHE/l1], [CACHE/L2], [CACHE/L3]

Key Name

Type

Description/Example

NumSets

(int)

Number of cache sets

Associativity

(int)

Associativity (ways)

LineSize

(int)

Line size (bytes)

TotalSize

(int)

Total cache size (bytes; L2/L3 only)

EnablePrefetch

(bool)

Hardware prefetcher (true/false)

EnableStatistics

(bool)

Enable statistics (true/false)

EnableCoherency

(bool)

SMP coherency protocol (true/false)

CoherencyProtocol

(string)

Protocol name ("MESI", "MOESI")

StatusUpdateInterval

(int)

Update window for stats

ReplacementPolicy

(string)

Line replacement policy ("MRU", "LRU", etc.)

EvictionThreshold

(int)

Eviction trigger (advanced)
