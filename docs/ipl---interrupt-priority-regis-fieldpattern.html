IPL - Interrupt Priority Register

Interrupt Priority Level (IPL) Field Pattern (Alpha AXP Processor Status Register)

 

•IPL is stored in bits 7:3 of the PS register (mask: 0xF8).

•All access and updates should use canonical helpers and masks for consistency.

 

Code Pattern  

Comments  

setIPL(ps, 7);  

Set IPL to 7 in variable 'ps'  

quint8 ipl = getIPL(ps);

Extract IPL value from 'ps'

iprBank.setIPL(15);

Set IPL atomically in IprBank

quint8 ipl = iprBank.getIPL();

Get current IPL from IprBank

setIPL(ps, value);  

Set IPL in 'ps' to variable 'value'

canAcceptInterrupt(quint8)

IPL comparison methods for interrupt logic - passed interrupt must a higher interrupt the current interrupt priority.

 

Typical pattern for extracting and setting:

    quint8 ipl = getIPL(newPS);           // Extract IPL from a new PS value

    iprBank.setIPL(ipl);                  // Set IPL in the processor register

 

General Guidance:

•Use the symbolic mask and shift (e.g., PS_IPL_MASK, PS_IPL_SHIFT) for all bit operations.

•Avoid magic numbers—use project constants for readability and safety.

•Use member functions for thread safety and encapsulation when working with IprBank.
