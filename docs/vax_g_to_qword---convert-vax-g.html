VAX_G_TO_QWORD - Convert VAX G_Float to 64-bit signed integer

Convert VAX G_Float to 64-bit signed integer (QUADWORD).

•The macro does a simple, direct conversion—no reserved/overflow/inexact detection.

•Add additional exception/trap logic outside the macro as needed for full architectural fidelity.

•Converts VAX G_floating (raw bits) to int64_t (quadword integer).

•Uses dt_gfloat for full fidelity if available.

 

Considerations.

•This does not check reserved operands/NaN.

 

/*

 * Macro: VAX_G_TO_QWORD

 *

 * Converts VAX G_floating (raw 64-bit value) to a 64-bit signed integer (quadword).

 *

 * Usage:

 *   quint64 gBits = context->readFpRegRawGFloat(m_ra);

 *   qint64 result = 0;

 *   VAX_G_TO_QWORD(gBits, result);

 *

 * Arguments:

 *   g_bits    - The raw 64-bit G-float value (as quint64).

 *   qword_out - A reference/variable to receive the converted 64-bit signed integer (qint64).

 *

 * How it works:

 *   - Wraps the raw bits in a dt_gfloat object.

 *   - Calls dt_gfloat::toDouble() for conversion to IEEE double.

 *   - Casts the double value to qint64 (truncating any fractional part).

 *   - No check is performed for reserved operands or NaNs.

 *   - No overflow checking—out-of-range G-float values will saturate or wrap per C++ cast rules.

 *

 * Limitations:

 *   - Does NOT raise Alpha/VAX reserved operand, overflow, or inexact exceptions.

 *   - If G-float is reserved/NaN, result is implementation-defined.

 *   - For full Alpha/VAX architectural compliance, add additional checks before/after macro use.

 *

 * Architectural Reference:

 *   - Alpha System Architecture Reference Manual, Ch. 4, Floating-Point Conversion Instructions.

 *   - VAX G_float → Integer conversion rules.

 *

 * Example:

 *   // In an Alpha grain:

 *   quint64 gBits = context->readFpRegRawGFloat(m_ra);

 *   qint64 qResult = 0;

 *   VAX_G_TO_QWORD(gBits, qResult);

 *   context->writeIntReg(m_rc, qResult);

 */
