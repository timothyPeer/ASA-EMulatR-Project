Legend:

 

•x = supported/used on that platform.

•x* = supported on SRM-console Alpha systems (typical for Linux/BSD on SRM).

•x† = present/used when the specific hardware/firmware implements the feature (e.g., perfmon or extended fields).

 

Key implementation references in your source:

•The read/write switch handling for many of the above (ASN, ASTEN read, ENTRY_0..7, EXC_* set/get, FPCR, ICCSR, PERFMON_CTRL/MASK, INTERRUPT_ENTRY, IVPTBR, KGB, KSP, MCES, MEM_MGMT_ENTRY, MMCSR/W1, PAL_BASE, PANIC_ENTRY, PCR, PDR, PS, SIRR, SYSCALL_ENTRY, TEB_BASE, VA, WHAMI) is in IprBank::readIpr/writeIpr.

•Architectural defaults and perf/entry initialization (PERFMON_0..7, PERFMON_CTRL/MASK, ENTRY_0..7) are set in initializeDefaults().

•Backing fields and accessors for EXC_* and PERFMON/ENTRY arrays are in the header.

•As Registers are validated - the register name links to an implementation page for the register.
